# 制作ワークフロー（逆算方式）
*v1.0 / 2026-05-20*

---

## 標準スケジュール（公開日を0として逆算）

```
公開日 -21日  ┌─ 会議（企画テーマ出し・方向性決定）
             │
公開日 -18日  ├─ 企画立案（tea支援：競合調査・アイデア出し）
             │
公開日 -15日  ├─ 企画案決定（チーム内レビュー・承認）
             │
公開日 -13日  ├─ 企画書作成（tea支援：企画書自動生成）
             │
公開日 -11日  ├─ 絵コンテ作成（tea支援：絵コンテ自動生成）
             │
公開日  -9日  ├─ 撮影アポイント（出演者・場所・機材確認）
             │
公開日  -6日  ├─ 撮影
             │
公開日  -3日  ├─ 編集締め切り
             │
公開日  -1日  ├─ 上司チェック（修正対応含む）
             │
公開日   0日  └─ 動画公開 🚀
```

> 🛠 **ツール:** `tools/schedule_tool.html` を開いて公開日を入力→自動計算

---

## 各工程の詳細

### 1. 会議（-21日）
**目的:** 月次コンテンツ方針の確認・個別案件の方向性決定
**アジェンダ:**
- 先月KPT振り返り
- 今月のテーマ・投稿本数決定
- 各案件の担当割り当て

**teaへの依頼例:**
```
「今月のコンテンツ会議のアジェンダ作って。
 先月のリーチが低かったので改善策も入れて」
```

---

### 2. 企画立案（-18日）
**目的:** ターゲット・構成・フックを設計する
**teaへの依頼例:**
```
「TikTok採用動画の企画立案して。
 ターゲット：25〜35歳・転職検討中・未経験OK
 競合との差別化も含めて」
```
**成果物:** 企画案メモ（`planning/` に保存）

---

### 3. 企画案決定（-15日）
**目的:** チーム全員で企画をレビューし、GOサインを出す
**チェックポイント:**
- [ ] ターゲットに刺さる内容か
- [ ] 撮影が現実的に可能か
- [ ] LINEへの誘導が設計されているか
- [ ] 日の丸交通のブランドイメージに合っているか

---

### 4. 企画書作成（-13日）
**目的:** 企画の詳細を文書化。社内共有・上司報告に使う
**teaへの依頼例:**
```
「企画案[タイトル]の企画書を
 planning_template.mdのフォーマットで作って」
```
**成果物:** `planning/[案件名]_企画書.md`

---

### 5. 絵コンテ作成（-11日）
**目的:** カット割り・テロップ・BGMを設計。撮影・編集の指示書になる
**teaへの依頼例:**
```
「[企画書]を元に60秒の絵コンテ作って。
 フックパターンはB（共感系）で」
```
**成果物:** `storyboards/[案件名]_絵コンテ.md`

---

### 6. 撮影アポイント（-9日）
**確認事項:**
- 出演者の日程確認
- 撮影場所の確保（社内 or 外部）
- 機材・衣装・小道具の準備リスト
- 台本の最終確認（出演者への共有）

---

### 7. 撮影（-6日）
**撮影当日のチェックリスト:**
- [ ] 絵コンテ印刷 or スマホで表示
- [ ] バッテリー・SDカード残量確認
- [ ] 音声チェック（ノイズ・BGM混入）
- [ ] 撮影後：素材をクラウドにバックアップ

---

### 8. 編集締め切り（-3日）
**編集指示書の基本:**
- 尺を守る（TikTok: 15〜60秒 / Instagram: 15〜90秒）
- 冒頭3秒に最大フックを置く
- テロップ：読みやすいフォント・色
- 最後必ずLINEへのCTA

**teaへの依頼例:**
```
「[動画タイトル]の説明テキスト・ハッシュタグ考えて。
 プラットフォーム：TikTok / Instagram」
```

---

### 9. 上司チェック（-1日）
**提出物:**
- 完成動画ファイル
- 投稿テキスト・ハッシュタグ
- 投稿日時

**修正依頼があった場合:** 当日中に対応→再提出

---

### 10. 動画公開（0日）
**公開後すること:**
- 投稿URLをチームに共有
- コメント・DMを確認（初動24時間が重要）
- KPIシートに投稿記録を追加

---

## 緊急・短納期の場合のショートフロー

スケジュールが取れない場合の最短版（10日以内）:

```
会議+企画案決定（1日）→ 絵コンテ（1日）→ 撮影（1日）→ 編集（2日）→ 上司チェック（1日）→ 公開
計：7日
```
> ※ 短納期は品質リスクあり。月1回以内を目安に
