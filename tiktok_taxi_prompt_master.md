## 📌 マスタープロンプト（全企画共通）

```
【背景】
あなたは日本のタクシー業界に精通した動画企画プロデューサーです。
以下の条件に基づいて、TikTok/Instagram Reels 用の企画案を作成してください。

【ターゲット】
- 職業変更を検討中の20〜50代男性
- 給与面での不満がある現職層
- 「タクシー運転手」という職業に興味はあるが、イメージが悪い層

【プラットフォーム仕様】
- 尺: 15〜60秒
- 形式: 縦型 9:16
- 音声: 著作権フリー or 人気トレンド曲
- テキスト: 冒頭フック（3秒以内）+ 本編 + CTA（LINE登録促進）

【バイラル要素（必須）】
1. 冒頭3秒で「え、そうなの？」という驚きor共感を作る
2. ドライバー側の本音 or 業界のあるある
3. 視聴者が「自分も同じ！」と思える内容
4. 参加型ハッシュタグキャンペーンの余地
5. 保存率を高める「ライフハック要素」or「情報提供」

【禁止事項】
- 給与額の具体的表記（イメージ低下）
- ネガティブなドライバー像（運転マナーなど）
- 長い説明（冒頭は結論から）
- 他業種との比較（給与など）

【成功事例の共通点】
- 日本TikTokで伸びるコンテンツ：ノスタルジア OR シュールユーモア OR 専門性×親しみやすさ
- タクシー運転手は「暗い職業」のイメージ → 「意外と楽しい・やりがいある」を見せる
- グループチャレンジなら「複数ドライバーの本音」という参加型にする
```

## 🎬 個別プロンプト① 「ドライバーのあるあるランキング」

### 【参考トレンド】「My Top 5 Horror Movies」

**プロンプト:**
```
以下の指示に従い、「タクシードライバーのあるあるランキング」動画企画を5案作成してください。

【フォーマット】
冒頭: 「タクシー運転手あるあるTOP5、一つでも当てはまったらシェア必須」
本編: ランキング形式で5つのあるあるを列挙（テンポよく、テキストオーバーレイ）
各あるあるの内容: 笑える + 共感性高い + 業界知識が伝わる内容

【あるあるの例】（必須で含める3つ以上）
- 乗客のマナー関連（無言電話、寝ながら乗る、など）
- 給与・労働時間の意外な事実（「夜勤の方が稼げる」など）
- 運転技術・経験の価値（「渋滞を最短ルートで抜ける」など）
- 業界ならではの人間関係（タクシー会社のドライバー同志の絆など）
- 女性ドライバーが増えている実態（多様性アピール）

【CTA】
「あなたのあるあるは？コメント欄で教えてください」
→ ハッシュタグ: #タクシー運転手あるある #給与UP #ドライバー求人

【推奨音源】
- アップテンポなコミック音楽
- TikTok人気: 「Everything I Wanted」など

【ハッシュタグ】
#タクシー運転手 #業界あるある #給与相談 #転職 #ドライバー募集
```

## 🎬 個別プロンプト② 「ビフォーアフター：隠れた職人技」

### 【参考トレンド】「World Stop」（ビフォーアフター）

**プロンプト:**
```
タクシー運転手の「隠れた職人技」をビフォーアフター形式で紹介する企画を5案作成してください。

【フォーマット】
冒頭3秒: 「え、そんなことしてたの？」という驚き
ビフォー: 一般的なタクシー利用のイメージ or 悪いドライバーの例
ターニングポイント: 「でも実は...」テロップ + 音楽の盛り上がり
アフター: 優秀なドライバーの技術・配慮が伝わる映像

【職人技の例】（最低3つ提案）
- 最短ルート選択（渋滞を回避する経験値）
- 乗客の快適性への配慮（温度調整、ルート説明など）
- 安全運転技術（雨の日の走行など）
- 女性客への安心感（防犯カメラ、丁寧な言葉遣い）
- 高齢者への配慮（ゆっくり走行、乗降サポート）

【CTA】
「こういう気遣いができるドライバーになりませんか？」
→ LINE登録へ誘導

【推奨音源】
- 「World Stop」的な盛り上がり感のトレンド曲
- または驚きを演出する効果音

【ハッシュタグ】
#タクシー運転手 #職人技 #ドライバー採用 #給与UP #やりがい
```

## 🎬 個別プロンプト③ 「ドライバーズボイス：業界の本音」

### 【参考トレンド】「You Know You Like It」（グループチャレンジ）

**プロンプト:**
```
複数のタクシードライバーが「業界の本音」を語るグループインタビュー企画を5案作成してください。

【フォーマット】
冒頭: 「現役ドライバーに聞いた、タクシー業界の本音」
本編: 4〜6人のドライバーが交互に本音を発言
- 「給与は思ったより...」
- 「人間関係は...」
- 「やりがいは...」
各ドライバーの発言はテキストオーバーレイ + 顔出し or シルエット

【登場人物の多様性（必須）】
- 男性ドライバー（40代）
- 女性ドライバー（30代）
- 新人ドライバー（20代）
- シングルファザー
- 副業層

【話すべき本音テーマ】
- 「給与：想像以上に稼げる」OR「実は選べる勤務」
- 「人間関係：ドライバー仲間の絆が強い」
- 「自由度：朝型・夜型を選べる」
- 「やりがい：乗客の感謝がダイレクト」
- 「ワークライフバランス：シフト自由」

【CTA】
「あなたも本音で働けるドライバーに。LINE登録で給与/勤務相談」

【推奨音源】
- 「You Know You Like It」的なグループ感のある曲
- またはトークがメインならBGM控えめ

【ハッシュタグ】
#タクシー運転手 #本音 #給与 #転職 #やりがい #ドライバー採用
```

## 🎯 使用方法

1. **NotebookLM / ChatGPT に以下をコピペ:**
   - マスタープロンプト + 個別プロンプト（1つ or 複数）

2. **AIの出力:**
   - 5企画案が生成される
   - 各企画: フック文 + 本編構成 + CTA + ハッシュタグ付き

3. **選定 → 台本化:**
   - 気に入った企画を選んで、「これを60秒の台本に起こしてほしい」と指示

4. **動画化:**
   - スタッフ or Kling AI で映像化
   - TikTok / Instagram に投稿
