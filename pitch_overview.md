# SNSマーケティング運用支援 — 活動説明資料
*日の丸交通株式会社 担当上司向け / v1.0 / 2026-05-20*

---

## 1. なぜSNS採用に取り組むのか

**背景・課題**
- 交通業界全体で求人応募数が減少傾向
- 従来の求人媒体（Indeed・リクナビ等）だけでは、求職者との接点が限られる
- 特に20〜35歳層はSNSで情報収集する習慣が定着している

**私たちの答え**
> 「日の丸交通の魅力を、SNSで届ける仕組みを作る」

求人票に頼らず、日常的な発信で**「知る→興味を持つ→応募したくなる」**という流れを作る。

---

## 2. 何をやっているか

### 集客〜採用の全体フロー

```
SNS投稿（TikTok / Instagram / X）
    ↓
プロフィールのLINE誘導リンクをタップ
    ↓
LINE公式アカウント登録
    ↓
職場体験会 / 会社説明会 への案内
    ↓
応募代行・面接設定サポート
    ↓
入社
```

### 具体的な活動内容

| カテゴリ | 内容 |
|---|---|
| **コンテンツ制作** | 採用ショート動画・職場紹介リール・ドライバーインタビュー等 |
| **SNS運用** | TikTok・Instagram・X の投稿管理・コメント対応 |
| **LINE管理** | 登録者への自動・手動返信・情報配信・応募サポート |
| **効果測定** | 週次KPI確認・月次レポート作成 |
| **企画立案** | 月次コンテンツカレンダー・キャンペーン企画 |

## 3. KPI（追っている数字）

| 指標 | 意味 | 年間目標 |
|---|---|---|
| **SNSリーチ数** | 動画・投稿が何人に届いたか | 月10万リーチ |
| **LINE登録者数** | 採用に興味を持った人の数 | 年間40名 |
| **LINE応募者数** | 実際に応募意向を示した人数 | 年間20名 |
| **入社者数** | SNS経由の採用決定数 | 年間4名 |

> ※ LINE登録→応募の転換率50%、応募→入社の転換率20%を目標に設計

## 5. 今後の展開

### 短期（〜3ヶ月）
- SNSアカウントの投稿リズムを安定させる（週3本以上）
- LINE登録の動線を最適化（SNS→LINEの誘導率向上）
- 職場体験会・説明会の定期開催化

### 中期（〜6ヶ月）
- ショート動画のバイラル施策（人気投稿の型を確立）
- LINE登録者のセグメント管理（未経験者・有資格者・転職検討中など）
- 月次採用実績の安定化

### 長期（1年〜）
- 採用コスト比較（従来媒体 vs SNS経由）の可視化
- SNSフォロワー5,000人達成
- このモデルの横展開・標準化
