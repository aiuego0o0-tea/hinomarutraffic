## 学習の考え方

> 「やりながら学ぶ」を基本に。  
> 失敗はチームの資産。気づいたことはすぐ記録する。

**teaは学習パートナーでもある。**  
「これどういう意味？」「なんでこの投稿は伸びたの？」など、  
何でもLINEで聞けば答える。

## 振り返り習慣（チームで毎月）

### 月次KPT
毎月末のチームミーティングで実施。teaがファシリテーションをサポートする。

```
Keep（続けること）
・うまくいっていること、続けたいこと

Problem（やめること・改善すること）
・うまくいっていないこと、課題

Try（来月試すこと）
・新しく挑戦すること（具体的に1〜3個）
```

**teaへの依頼例:**
```
「今月のKPT振り返りをサポートして。
 先月やったこと：[内容]
 数値：[LINE登録数・リーチ数]」
```

## 「わからない」をすぐ解決する方法

チームメンバーがLINEでteaに聞けること：

| カテゴリ | 質問例 |
|---|---|
| SNS知識 | 「エンゲージメント率って何％あれば良いの？」 |
| 制作 | 「縦型動画のベストな字幕の大きさは？」 |
| マーケ | 「ペルソナって何？どうやって作る？」 |
| 法律・規制 | 「採用動画で言ってはいけない表現はある？」 |
| 文章 | 「この投稿文、もっとキャッチーにして」 |
| 企画 | 「今週の投稿ネタ3つ出して」 |

> **大切なのは「調べるより先にteaに聞く」習慣をつけること。**  
> 時間を節約して、考えることに集中できる。
